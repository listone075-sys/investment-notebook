# 投资分析项目 — AI Touzi

## 📌 项目定位

以 **价值投资为主、题材投资为辅** 的投资分析项目，核心思路：

| 流派 | 来源 | 策略要点 |
|------|------|---------|
| **价值投资** | 巴菲特、段永平 | 好生意 + 好公司 + 好价格；买股票就是买公司；长期持有；能力圈 |
| **题材投资** | 市场主题驱动 | 政策风口 + 产业趋势 + 事件催化；中短期波段操作 |

## 🧠 核心纪律

1. **不懂不做** — 只在自己的能力圈内做投资
2. **安全边际** — 好公司也要有好价格
3. **长期视角** — 以持有 3-5 年的视角审视标的
4. **独立思考** — 不跟风、不盲从
5. **纪律执行** — 严格按照检查清单和风控规则操作

## 🚀 使用方法

本仓库配合 Claude AI 使用，通过自然语言交互进行分析：
- `分析 [股票代码]` — 对标的进行价值投资分析
- `追踪 [题材名称]` — 追踪某个投资主题
- `复盘 [时间段]` — 回顾期间操作和收益
- `风险检查` — 检查当前组合风险状况

---

*本仓库仅为个人投资研究辅助，不构成投资建议。*
