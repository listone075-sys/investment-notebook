# 辅助脚本

> 用于数据采集、分析计算的 Python/Shell 脚本。

## 规划中的脚本

- [ ] `fetch_financials.py` — 爬取财报数据
- [ ] `valuation_calc.py` — DCF 估值计算器
- [ ] `portfolio_tracker.py` — 组合跟踪
- [ ] `sentiment_analysis.py` — 舆情分析

## 使用方式

```bash
# 初始化 Python 环境
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```
