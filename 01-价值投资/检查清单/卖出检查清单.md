# 价值投资 — 卖出检查清单

> 卖出比买入更难。不因为有更好的机会而卖，不因为涨多了而卖。

---

## 卖出理由

### ✅ 应该卖出
- [ ] **基本面永久恶化**：护城河被摧毁 / 核心产品被替代 / 竞争优势不再
- [ ] **管理层失信**：财务造假 / 严重损害股东利益
- [ ] **价格极度高估**：PE/PB 处于历史 90% 分位以上，且无合理逻辑支撑
- [ ] **发现明显更好的机会**：安全边际更大、确定性更高（换仓）

### ❌ 不应该因为以下理由卖出
- [ ] 股价涨了很多（除非已极度高估）
- [ ] 股价跌了很多（除非基本面恶化）
- [ ] 别人说该卖了
- [ ] 想腾出资金买别的（除非符合上面"应该卖"条件）
- [ ] 担心宏观经济（除非影响到公司基本面）
