# 标的分析报告模板

> **使用说明**：以此文件为模板生成新的标的分析 HTML 报告。
> 参考报告：`600801-华新建材-2026-07-15.html`（A 版格式）

## 格式规范

- **背景色**：浅灰底 `#f5f6fa`
- **卡片**：白色圆角卡片 `border-radius: 12px`
- **强调色**：`#2563eb`（蓝）
- **评级区**：顶部独立 verdict-box 卡片
  - 买入 → 绿色渐变底 + 绿字 ✅ 买入
  - 观察 → 黄色渐变底 + 黄字 ⚡ 观察
  - 回避 → 红色渐变底 + 红字 ❌ 回避
- **折叠**：卡片标题可点击折叠（JS toggle）
- **全部折叠**：顶部 "全部展开/全部折叠" 按钮

## 报告结构（10 个章节）

1. **核心速览** — 关键数据 grid-4 展示
2. **生意理解** — 公司速览 + 一句话怎么赚钱 + 核心护城河 + 商业模式拆解表 + 关键财务特征 + 主营业务构成 + 护城河量化评估
3. **管理层评估** — 核心人物 + 评估维度评分
4. **财务分析** — 五年核心指标 + 最新季度数据
5. **竞争格局** — 行业对比表 + 核心差异化分析
6. **段永平"三问"测试** — 能力圈 / 护城河 / 安全边际
7. **估值分析** — 当前估值 + 多方法估值（PE/PB/DCF/分部）
8. **催化剂时间线** — 未来 12 个月关键事件
9. **操作区间与仓位** — 买入/合理/偏高/高估四档 + 建仓节奏 + 卖出条件
10. **风险提示** — 风险表 + 最大风险情景

## CSS 结构

```css
:root {
  --bg: #f5f6fa;           /* 页面背景 */
  --card: #ffffff;          /* 卡片背景 */
  --card2: #f0f2f8;         /* 卡片辅助色 */
  --accent: #2563eb;        /* 强调色 */
  --accent-light: #dbeafe;  /* 强调色淡 */
  --green: #16a34a;         /* 买入/正面 */
  --green-bg: #dcfce7;
  --red: #dc2626;           /* 回避/负面 */
  --red-bg: #fee2e2;
  --yellow: #d97706;        /* 观察/中性 */
  --yellow-bg: #fef3c7;
  --text: #1e293b;          /* 正文 */
  --text2: #475569;         /* 次要文字 */
  --muted: #94a3b8;         /* 弱化文字 */
  --border: #e2e8f0;        /* 边框 */
}
```

## 关键 CSS 类

| 类名 | 用途 |
|------|------|
| `.card` | 内容卡片 |
| `.card h2` | 可折叠标题（onclick toggleSection） |
| `.section-body.collapsed` | 折叠状态 |
| `.verdict-box` | 评级卡片（渐变底） |
| `.grid-2/3/4` | 栅格布局 |
| `.stat-tile` | 数据瓦片 |
| `.key-insight` | 重点提示框（蓝/绿/黄/红） |
| `.tag-green/yellow/red/blue` | 标签 |
| `.tl-item` | 时间线条目 |
| `.q-item` | 三问测试条目 |

## 文件命名

`[代码]-[名称]-[日期].html`，如 `600801-华新建材-2026-07-15.html`

## 内容要求

每个报告必须包含：
- 竞争格局分析（至少一张对比表）
- 段永平"三问"测试
- 未来 12 个月关键催化时间线
- 多方法估值（PE/PB/DCF/分部）
- 具体的操作区间（买入/观察/高估）
- 仓位和建仓节奏
