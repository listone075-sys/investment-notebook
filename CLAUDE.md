# 投资分析项目 — Claude Context

## 项目概览

基于价值投资（巴菲特/段永平思想）+ 题材投资的投资分析系统。

## 核心投资原则

- **买股票=买公司**：分析标的第一性原理是"这家公司是否值得买下全部或部分"
- **能力圈**：只在理解透了的领域下注
- **安全边际**：好公司 + 好价格 = 好投资
- **市场先生**：市场是服务于你的仆人，不是指导你的导师
- **长期主义**：以年为单位持有，不试图预测短期波动

## 投资流程

### 价值投资分析流程
1. 理解生意（商业模式、护城河）
2. 评估管理层（诚信、能力、股东导向）
3. 分析财务（ROE、现金流、负债）
4. 估算内在价值（DCF、PE、可比估值）
5. 确定安全边际
6. 制定买入/卖出计划

### 题材投资分析流程
1. 确认题材级别（政策/产业/事件）
2. 判断持续性（短期/中期/长期）
3. 锁定核心受益标的
4. 制定进出策略
5. 设置止损止盈
